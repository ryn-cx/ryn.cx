## Overview

yt-dlapi is an unofficial YouTube API Python library built on top of
[yt-dlp](https://github.com/yt-dlp/yt-dlp). yt-dlapi was originally part of [Stream
Channeler](/projects/stream-channeler/), but was converted into a library to make it
easier to use in other projects.
