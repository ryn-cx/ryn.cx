## Overview

chirashi is an unofficial [Crunchyroll](https://www.crunchyroll.com/) API Python library
that wraps the Crunchyroll API and parses its raw JSON into typed
[Pydantic](https://docs.pydantic.dev/) models, giving you a small, structured API for
reading data about series, seasons, episodes, and search results.
