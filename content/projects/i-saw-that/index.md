## Overview

I Saw That is an automatic real time backup and versioning system written in Go. I Saw
That will automatically monitor a folder for changes and any time a change is detected
it will copy all of the files into a timestamped folder. I Saw That can be run from the
command line, or as a GUI application.

## Why

I Saw That was built to fill a niche need for something like
[AutoVer](https://beanland.net.au/AutoVer/) that could be run from the command line.

## Project Status

Somewhat abandoned. There is no longer a need for this software, so it is no longer
actively used or developed.
